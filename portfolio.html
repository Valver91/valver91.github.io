Generador QR's

Este programa utiliza las bibliotecas tkinter y qrcode para generar códigos QR a partir de un enlace proporcionado por el usuario en una interfaz gráfica. Al hacer clic en el botón "Generar QR", se crea y guarda el código QR correspondiente como una imagen en el sistema.

Control de asistencia

Este programa utiliza varias bibliotecas para realizar reconocimiento facial y registrar la asistencia de empleados. A continuación, explico el funcionamiento general y cómo se modifican los archivos Excel.

El programa utiliza las siguientes bibliotecas:

  • cv2 (OpenCV): Utilizada para trabajar con imágenes y la captura de video de la cámara.
  • face_recognition: Proporciona funciones para el reconocimiento facial, como la detección y codificación de caras.
  • os: Utilizada para acceder al sistema operativo y administrar archivos y directorios.
  • numpy: Utilizada para realizar operaciones numéricas y manipulación de matrices.
  • datetime: Utilizada para obtener la fecha y hora actual.
  • openpyxl: Proporciona funcionalidades para trabajar con archivos de Excel (xlsx).

El programa comienza leyendo las imágenes de los empleados desde una carpeta y las almacena en una lista llamada "mis_imagenes". También guarda los nombres de los empleados en una lista llamada "nombres_empleados".

En el enlace de "Código" podrá acceder al programa del proyecto.

ClearCut

El programa utiliza tkinter para crear una interfaz gráfica. La biblioteca rembg se utiliza para eliminar el fondo de una imagen. Al seleccionar un archivo de entrada y un archivo de salida, el programa utiliza rembg para procesar la imagen de entrada y guardarla con el fondo eliminado en el archivo de salida. La biblioteca rembg es útil para tareas de edición de imágenes que requieren la eliminación del fondo de forma automatizada.
